【前驅之王】葡萄牙挑機　本田Civic Type R再創圈速

本田（Honda）Civic Type R這部前驅揭背小跑車，外國已經有facelifted諜照流出，不過現有車款仍然四出挑機。今年內，繼英國Silverstone、比利時Spa-Francorchamps、匈牙利Hungaroring之後，剛剛去到葡萄牙Estoril賽道。

廠方剛發佈了足本影片，由車廂角度看看這部Type R如果創造紀錄。這條全長4.2公里的賽道曾經是方程式賽車場，今次Civic Type R由本田WTCR車手Tiago Monteiro駕駛，就創出2:01.84這個量產前驅車的圈速紀錄。
