【國產手機】潛望式鏡頭大比拼　NEX實淨 vs FIND X夢幻

潛望鏡頭是手機界新興起的名詞！有需要使用時，就會在機頂升上來影相拍片，兩個國產機品牌，分別在多星期前及剛剛推出了新機，它們分別是VIVO NEX及OPPO FIND X。肥通想問的是，用咗潛望鏡係咪影相會靚啲？道理上是說不通！但拿上手時係咪型啲？這是觀點與角度，買它不用說道理吧。

剛抵手機場的OPPO FIND X，可說是近期熱爆手機！OPPO行貨未有任何聲氣，暫時得到資料只會以水貨形式到港。這部機的潛望式鏡頭好特別，就是前置自拍鏡及機背的主鏡頭，在用家未開啟相機功能時，是會收藏在機身之內的，所以這時全部手機除了機側的按鍵，就是前後滑溜的機身，這可是智能手機在設計上的一個小突破；到你需要使用鏡頭拍照時，橫跨整個機頂的鏡頭部件，就會輕輕上升約1cm，前、後置鏡頭這時就會突出來供你拍照。

潛望鏡的設計真的有需要嗎？三禾電氣老闆Simon：「很多手機買家都覺得這個設計非常有型！實際用途嗎？第一點是可以保護鏡頭的鏡片，不會輕易弄花；其次有些用機較小心的用家，他們亦說驚手機給黑客入侵，會被開啟鏡頭功能偷拍，不過這個情況非常罕有，第一你要值得被人偷拍，其次你的相簿內已有不少相片吧？入侵後也不用遙距開相機拍你吧！不過使用潛望式鏡頭帶來的好處，是屏幕頂再不用留個M字額放鏡頭了，亦令機身的屏佔率高達94%。」較早一個星期推出的VIVO NEX都有潛望式鏡頭，不過只限自拍用的前置鏡頭，機背的主鏡頭還是留在機背之上。

兩機都針對自拍功能，我們就以前置鏡頭的成像能力作比較吧！驟看兩機鏡頭「可拍角度」都差不多一樣，但論人物膚色，VIVO NEX會靚一點、真實感重一點！相對OPPO FIND X會比較偏紅一點，但如果是女性用家，有可能喜歡吧，因為拍出來的膚色會更紅潤、更美麗！另外顏色的真實性方面，我留意到試拍相後的綠葉，VIVO NEX會真實一點，OPPO FIND X的綠色有點帶螢光，不太真實！

Simon續稱：「VIVO這個品牌偏向成熟及男性路線，所以相機的『味精』相對會少一點，相對OPPO這個品牌，是會較偏向年青路線，所以自拍照帶點夢幻感，及大家所謂的『味精』會重一點。」不說不知，其實兩間廠都有一點連帶性關係：「其實近年在手機界走紅的VIVO、OPPO及One Plus是三間兄弟公司，因為大家的母公司都是步步高。」

兩部手機還有一個相似點，就是叉電特快！兩機使用了超快叉電技術，只需配合兩機的原裝火牛及接線，叉電會十分快速，Simon說：「OPPO FIND X使用5V/4A叉電，而VIVO NEX則可使用10V / 2.25A叉電，兩機也可以做到0-100%叉電，在約半小時內完成；就算用普通火牛叉電，源於兩機電池設計很特別，將一枚電池分成兩部份，就算用普通火牛，也可做到0-100%，在30-40分鐘內完成。」

再試是解鎖功能，先來OPPO FIND X，只要按開關鍵啟動屏幕，再用手指由下而上一掃，手機前置鏡頭就會自動升起，肥通測試時鏡頭開啟後不用半秒，就已完成整個人臉辨識程序了，但經常升上升落，會否很易弄壞？Simon說：「OPPO官方表示，這個潛望鏡式組件，基本可以做到5萬次開合而不會發生問題，如果算上你每日用上百多次，起碼都可以用5年呢！」至於VIVO NEX的解鎖方法，都用了不太多手機使用的「屏幕指紋解鎖」：「雖然近幾年，屏幕指紋解鎖大家經常都聽到，不過實質香港市場上買到的機種，就只有VIVO出過的X20、X21及PORSCHE DESIGN HUAWEI Mate RS，而最新的就是NEX了！所以對不少用家來說，都還是一件新鮮事物。」

最後總結，兩部都不錯，可以買哪一部呢？就肥通個人而言，是喜愛VIVO NEX多一點的，感覺整體設計非常穩定成熟，可以真心媲美不少大牌的旗艦級手機了，而且屏幕顏色實淨、相片成像亦實淨，兩機中實用性會強一點，最重要是價錢亦較平一點（6GB+128GB $4,400、8GB+128GB $5,700、8GB+256GB $6,000）。至於OPPO FIND X，買它的最主要原因是外形夠型！非常有玩味，機前機後沒有一點多餘之物，手感非常滑溜，是一種新的體驗，不過機價則偏貴一點，暫時只有8GB+128GB（$6,380）一個版本，比256GB版的NEX還要高價呢！

查詢：三禾電氣（旺角亞皆老街83號先達廣場1樓F33）
